# Contributing to @nestbolt/translatable

Thank you for considering contributing! Here's how you can help.

## Getting Started

1. Fork the repository
2. Clone your fork:
   ```bash
   git clone https://github.com/<your-username>/translatable.git
   cd translatable
   ```
3. Install dependencies:
   ```bash
   pnpm install
   ```
4. Create a feature branch:
   ```bash
   git checkout -b my-feature
   ```

## Development

### Build

```bash
pnpm run build
```

### Run Tests

```bash
pnpm test
```

### Watch Mode

```bash
pnpm test:watch
```

### Lint

```bash
pnpm run lint
```

## Pull Request Guidelines

- Create a feature branch from `main`.
- Keep PRs focused — one feature or fix per PR.
- Make sure all tests pass before submitting.
- Add tests for any new functionality.
- Follow the existing code style and patterns.

## Reporting Issues

- Use [GitHub Issues](https://github.com/nestbolt/translatable/issues) to report bugs or request features.
- For security-related issues, use the **security** label.
